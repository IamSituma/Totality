# Totality

Monorepo containing the frontend and backend for Totality Logistics.

## Structure

- `frontend/` – React/Next.js app for the waitlist interface.
- `backend/` – Node.js/Express server handling waitlist operations, email notifications, and database.

## Setup

### Backend
```bash
cd backend
npm install
npm run dev

### Frontend
```bash
npm install
npm run dev or npm start
